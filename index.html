Installation

Самый быстрый способ создать проект на Vue 3 с использованием Vite:

# NPM
npm create vue@latest

# Yarn
yarn create vue

# PNPM
pnpm create vue

Или подключение через CDN (для прототипирования): 

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
⚙️

Configuration

Global Config (main.js)

import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

// Глобальные свойства
app.config.globalProperties.$api = '...'

// Подключение плагинов
app.use(Router)
app.use(Pinia)

app.mount('#app')

Markdown Rendering

Использование библиотеки marked для рендеринга.

<template>
  <div v-html="renderedMarkdown"></div>
</template>

<script setup>
import { computed } from 'vue'
import { marked } from 'marked'

const props = defineProps(['content'])

const renderedMarkdown = computed(() => {
  return marked.parse(props.content)
})
</script>
🎨

CSS Configuration

Vue 3.2+ позволяет использовать реактивные значения прямо в CSS через v-bind.

Script Setup
<script setup>
import { ref } from 'vue'
const theme = ref({
  color: 'red',
  text: 'hello!'
})
</script>
Style Scoped
<style scoped>
.text {
  /* Привязка к переменной JS */
  color: v-bind(theme.color);
}
h1 {
  content: v-bind(theme.text);
}
</style>

Emitted Values

Коммуникация от дочернего компонента к родителю.

Child Component
<script setup>
const emit = defineEmits(['update', 'close'])

function handleClick() {
  // Отправка события с данными
  emit('update', { id: 1, name: 'Vue' })
}
</script>

<template>
  <button @click="handleClick">
    Update Parent
  </button>
</template>
Parent Component
<template>
  <ChildComponent 
    @update="handleUpdate" 
    @close="isOpen = false" 
  />
</template>

<script setup>
const handleUpdate = (data) => {
  console.log('Received:', data)
}
</script>

Errors Management

Глобальная обработка ошибок и перехват внутри компонентов.

Global Handler (main.js)
app.config.errorHandler = (err, instance, info) => {
  // Отправка в сервис мониторинга (Sentry, etc.)
  console.error('Global Error:', err)
  console.log('Component:', instance)
  console.log('Info:', info)
}
Component Level (errorCaptured)
<script setup>
import { onErrorCaptured } from 'vue'

onErrorCaptured((err, instance, info) => {
  // Логика обработки ошибки потомка
  console.log('Captured:', err)
  
  // Вернуть false, чтобы ошибка не всплывала выше
  return false 
})
</script>
